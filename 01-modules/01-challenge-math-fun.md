# Math Fun Challenge

Create a node module able to do the following operations:

#### math.js

- add(a, b) => a + b
- subtract(a, b) => a - b
- multiply(a, b) => a * b;
- divide(a, b) => a / b;
- random(a, b) => returns a random number between a and b

Then import and use the `add` and `multiply` methods in `fun.js`.
